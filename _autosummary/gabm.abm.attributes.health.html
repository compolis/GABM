class gabm.abm.attributes.health.HealthMap

Bases: GABMAttributeMap

A mapping of HealthIds to Health.

By default, the map is initialized as follows:

h0 = HealthID(0)
h1 = HealthID(1)
h2 = HealthID(2)
h3 = HealthID(3)
h4 = HealthID(4)
h5 = HealthID(5)
items: Dict[HealthID, Health] = {
    h0: Health(h0, "unknown"),
    h1: Health(h1, "very good"),
    h2: Health(h2, "good"),
    h3: Health(h3, "fair"),
    h4: Health(h4, "bad"),
    h5: Health(h5, "very bad")
}
super().__init__(items)
