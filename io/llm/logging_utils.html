Logging Utils Module

Centralized logging setup utility for GABM LLM modules. Import and call setup_module_logger(__name__, log_file_name) at the top of each module/script.

src.io.llm.logging_utils.setup_module_logger(module_name, log_file_name)

Set up a logger for a module, writing to data/logs/log_file_name and the console. Returns a logger instance for use in the module.
